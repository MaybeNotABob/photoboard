## Photoboard

Display an organisation's personnel in a structured hierarchy tree. &nbsp;Photographs of staff members are accompanied by their role/title and name.

```
              +---------+              
              |         |              
              |         |              
              |         |              
              |         |              
              +---------+              
                 BOSS                  
               JANE DOE                                                               
+---------+   +---------+   +---------+
|         |   |         |   |         |
|         |   |         |   |         |
|         |   |         |   |         |
|         |   |         |   |         |
+---------+   +---------+   +---------+
   I.T.          CARGO         SALES   
 JOHN DOO       JILL GROW     JACK WOE 
```
